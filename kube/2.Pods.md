### livenessProbe 예제 (살아 있는지 확인)

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: nginx-lp
  labels:
    type: app
spec:
  containers:
  - name: app
    image: nginx
    livenessProbe:
      httpGet:
        path: /not/exist
        port: 80
      initialDelaySeconds: 5
      timeoutSeconds: 2 # Default 1
      periodSeconds: 5 # Defaults 10
      failureThreshold: 1 # Defaults 3
```

### readinessProbe 예제 (준비가 되었는지 확인)

### health check 예제

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: nginx-health
  labels:
    type: app
spec:
  containers:
  - name: app
    image: nginx
    livenessProbe:
      httpGet:
        path: /
        port: 2005
    readinessProbe:
      httpGet:
        path: /
        port: 2005
```

### multi container 예제

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: nginx-redis
  labels:
    type: stack
spec:
  containers:
  - name: app
    image: nginx
    env:
    - name: REDIS_HOST
      value: "localhost"
  - name: db
    image: redis
```
