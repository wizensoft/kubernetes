# Deployment

Deployment using replicaset

## 예제

### 기본 예제

kube/4/deploy.yaml

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: deploy
spec:
  replicas: 3
  selector:
    matchLabels:
      type: app
      service: nginx
  template:
    metadata:
      labels:
        type: app
        service: nginx
    spec:
      containers:
      - name: nginx
        image: nginx
        livenessProbe:
          httpGet:
            path: /
            port: 80
```

```
k set image deploy/deploy nginx=nginx
k apply -f deploy.yaml
k get rs -w
k describe deploy/deploy
k rollout history -f deploy.yaml
k set image deploy/deploy nginx=nginx --record=true
k rollout history -f deploy.yaml
k rollout history -f deploy.yaml --revision=2
k rollout status deploy/deploy
k rollout undo deploy/deploy
k rollout undo deploy/deploy --to-revision=3
```

### 추가 예제

kube/4/deploy-strategy.yaml

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: deploy
spec:
  replicas: 3
  selector:
    matchLabels:
      type: app
      service: nginx
  minReadySeconds: 5
  strategy:
    type: RollingUpdate
    rollingUpdate:
      maxSurge: 1
      maxUnavailable: 1
  template:
    metadata:
      labels:
        type: app
        service: nginx
    spec:
      containers:
      - name: nginx
        image: nginx
        livenessProbe:
          httpGet:
            path: /
            port: 80
```

```
k describe deploy/deploy
k set image deploy/deploy nginx=nginx
k get rs -w
```
